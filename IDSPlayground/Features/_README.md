# Features

Add your flow folders here. Each flow gets its own subfolder.

See **FLOW_TEMPLATE.md** in the project root for step-by-step instructions.

```
Features/
└── YourFlow/
    ├── YourFlowFlow.swift
    ├── YourFlowViewModel.swift
    └── Views/
        ├── FirstScreenView.swift
        └── SecondScreenView.swift
```
